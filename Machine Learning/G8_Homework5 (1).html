4.5 Why do you think all the coefficients of Lasso and ElasticNet models are zero?

The alpha values for the models have not been optimized. As such they default to 1 and this is a high value for alpha, making it so all the terms are penalized. This causes the model to become 0 for lasso, and consequently elastic net aswell.

As can be seen in all the graphs, lower values of alpha make it so that coefficients are not penalized as heavily. This makes it so most of the coefficents are "allowed" to remain in the model. As alpha increases, the penalty tacked onto the coefficients increases as well making it harder for features to stay in the model. In the Lasso and Elastic net case, this forces them to 0, or drops them. In the Ridge case, the coefficients just approach 0.
